CATIA is a solid modeling program developed by Dassault Systemes. Among the famous architects extensively using this system, one can name the Pritzker Prize winner - Frank Gehry. Recently, Dassault Systemes has significantly expanded the range of tools offered to architects and builders. CATIA automation can be done either through xGen or through a standard COM interface.

COM is a component interface for application interaction. It is through him that software systems open access to their API. Despite the fact that this is a rather mature technology, it is very poorly documented, all information must be extracted directly from the code. Many programmers use scripting languages ​​built into applications. Such languages ​​are interpreted and do not offer sufficient performance necessary for numerical calculations. We, in turn, are developers of the Shape Factory system, which allows us to automate COM applications using the high-performance technical computing language Julia. Thus, we suggest that you forget about all the problems associated with interpreted programming languages.

We can say that we turned .Net inside out to connect two different memory management systems - C # and Julia. Now we have no barriers to develop high-performance technical calculations in this environment.
